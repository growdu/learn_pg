## Task 1: 后端 - ServeClusterOverview 改为 GET /api/cluster/:id/overview

**Files:**
- Modify: `backend/internal/api/handler.go`

- [ ] **Step 1: 修改 ServeClusterOverview**

当前 `ServeClusterOverview` 是 POST，接收 `requestNodes[]` 含完整凭据。改为 GET /api/cluster/:clusterId/overview。

从 URL 提取 clusterId，从 workspace 找到 cluster，对每个 node 调用 `InspectNode` 获取状态。

当前 `inspectClusterNode` 接收 `clusterNodeRequest`（含 host/port/user/password）。改为接收 `nodeId` string，从 workspace 读取配置后连接。

```go
// ServeClusterOverview handles GET /api/cluster/{clusterId}/overview
func (h *Handler) ServeClusterOverview(w http.ResponseWriter, r *http.Request) {
    if r.Method != http.MethodGet {
        h.writeError(w, r, http.StatusMethodNotAllowed, "GET required")
        return
    }

    // Extract clusterId from URL path: /api/cluster/{clusterId}/overview
    clusterID := extractClusterIDFromPath(r.URL.Path) // implement this
    if clusterID == "" {
        h.writeError(w, r, http.StatusBadRequest, "cluster id required")
        return
    }

    // Find cluster in workspace
    projects, _, err := h.workspace.readSnapshot()
    if err != nil {
        h.writeError(w, r, http.StatusInternalServerError, err.Error())
        return
    }
    var targetCluster *workspaceCluster
outer:
    for pi := range projects {
        for ci := range projects[pi].Clusters {
            if projects[pi].Clusters[ci].ID == clusterID {
                targetCluster = &projects[pi].Clusters[ci]
                break outer
            }
        }
    }
    if targetCluster == nil {
        h.writeError(w, r, http.StatusNotFound, "cluster not found")
        return
    }

    resp := clusterOverviewResponse{
        Success:   true,
        Timestamp: time.Now().Unix(),
        Nodes:     make([]clusterNodeStatus, 0, len(targetCluster.Nodes)),
    }

    for i, node := range targetCluster.Nodes {
        // Get node status using nodeId - register + activate if needed
        status := h.inspectNodeByID(node.ID)
        if status.ID == "" {
            status.ID = fmt.Sprintf("node-%d", i+1)
        }
        resp.Nodes = append(resp.Nodes, status)
        resp.Summary.TotalNodes++
        if status.Connected {
            resp.Summary.ConnectedNodes++
        }
        if strings.EqualFold(status.ClusterType, "physical") {
            resp.Summary.PhysicalNodes++
        }
        if strings.EqualFold(status.ClusterType, "logical") {
            resp.Summary.LogicalNodes++
        }
    }

    writeJSON(w, r, http.StatusOK, resp)
}

// inspectNodeByID inspects a node by its ID, using connMgr for connection.
func (h *Handler) inspectNodeByID(nodeID string) clusterNodeStatus {
    // Find node config in workspace
    projects, _, _ := h.workspace.readSnapshot()
    var node workspaceNode
    for pi := range projects {
        for ci := range projects[pi].Clusters {
            for ni := range projects[pi].Clusters[ci].Nodes {
                if projects[pi].Clusters[ci].Nodes[ni].ID == nodeID {
                    node = projects[pi].Clusters[ci].Nodes[ni]
                }
            }
        }
    }
    if node.ID == "" {
        return clusterNodeStatus{ID: nodeID, Error: "node not found in workspace"}
    }

    status := clusterNodeStatus{
        ID:          node.ID,
        Name:        node.Name,
        Host:        node.Host,
        Port:        node.Port,
        Database:    node.Database,
        ClusterType: node.ClusterType,
        Role:        node.Role,
    }

    // Try to get existing connection from connMgr
    client, err := h.connMgr.Get(nodeID)
    if err != nil {
        // Try to activate using registered config
        h.connMgr.Register(nodeID, connection.Config{
            Host:     node.Host,
            Port:     node.Port,
            User:     node.User,
            Password: node.Password,
            Database: node.Database,
        })
        if err2 := h.connMgr.Activate(nodeID); err2 != nil {
            status.Error = err.Error()
            return status
        }
        client, _ = h.connMgr.Get(nodeID)
    }

    return h.inspectNodeClient(node, client)
}
```

Also add `extractClusterIDFromPath`:

```go
func extractClusterIDFromPath(path string) string {
    // /api/cluster/{clusterId}/overview or /api/cluster/{clusterId}
    parts := strings.TrimPrefix(path, "/api/cluster/")
    i := strings.Index(parts, "/")
    if i >= 0 {
        return parts[:i]
    }
    return strings.TrimSpace(parts)
}
```

## Task 3: 后端 - provision_discovery.go inspectClusterNode 使用 connMgr

**Files:**
- Modify: `backend/internal/api/provision_discovery.go`

- [ ] **Step 1: 检查 inspectClusterNode 用法**

`provision_discovery.go` 中的 `inspectClusterNode` 临时创建 `&pg.Client{}`。改为使用 `connMgr.Get(nodeId)` 或从 workspace 读取配置。

当前代码模式：
```go
client := &pg.Client{}
if err := client.Connect(host, port, user, password, db); err != nil {
    return status
}
defer client.Close()
```

应改为：
1. 尝试 `h.connMgr.Get(nodeId)`
2. 如果失败，从 workspace 读取配置
3. 注册并激活

这个改动比较复杂，因为 `provision_discovery.go` 是在 workspace 外操作。简单方案：保持当前临时连接模式，因为这是 provision 过程中的临时连接，不需要纳入 connMgr 管理。

**结论**：M3 阶段不改造 `inspectClusterNode`，它只在 provision 过程中使用，不影响前端观测接口。跳过此 Task。
