# pg 内核学习应用开发

需要实现一个web app，主要目的如下：

1. 辅助学习postgres内核；
2. 重点关注事务、并发、wal、clog等存储引擎相关机制原理，后续保留对sql引擎的扩展；
3. 从运行时重要数据结构的变更，内存和文件块的变化过程；
4. 动态的展示数据写入过程；
5. 动态的展示数据的读取过程；
6. 动态的展示事务的执行过程；
7. 动态展示常见sql的执行过程；

必须要时可以考虑ebpf技术。

前端语言实现尽量考虑跨平台，兼容桌面（ios、android、mac、win）、web。
后端api相关考虑使用go实现。
后端ebpf相关考虑使用rust实现。

综合根据项目需求进行选型。
